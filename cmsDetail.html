Customer Management System

The Customer Management System is a full-featured web app designed to help businesses efficiently manage their customer data. It supports operations like creating, reading, updating, and deleting customer records. Built using the MERN stack, it also includes a secure login system, responsive UI, and advanced search/filter features.

Key Features:

  • Customer profile management (CRUD)
  • Search and filter functionality
  • Role-based authentication
  • Responsive and user-friendly UI
  • MongoDB database integration

Technical Stack:

  • MongoDB
  • Express.js
  • JavaScript
  • Node.js
  • JWT Authentication
