Todo App

This Todo App is a simple yet powerful task manager built using modern React best practices. It helps users organize their daily tasks by adding, editing, deleting, and marking tasks as completed. State management is handled with Redux Toolkit for efficiency and scalability.

Key Features:

  • Add, edit, and delete todos
  • Mark tasks as completed
  • Filter todos (All, Completed, Pending)
  • Lightweight, responsive UI
  • Persisted state with Redux

Technical Stack:

  • React
  • Redux Toolkit
  • JavaScript (ES6+)
  • CSS
