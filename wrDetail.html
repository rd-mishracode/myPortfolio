Weather Report App

The Weather Report App is a responsive React.js application that fetches and displays real-time weather information using the OpenWeatherMap API. Users can search for any city and receive live weather conditions including temperature, humidity, wind speed, and overall conditions.

Key Features:

  • Search weather by city name
  • Real-time data from OpenWeatherMap API
  • Displays temperature, humidity, wind speed, and condition
  • Responsive and mobile-friendly UI
  • Error handling for invalid input

Technical Stack:

  • React.js
  • CSS
  • OpenWeatherMap API
  • JavaScript (ES6+)
